# Squad de Agentes

**150+ agentes IA especializados que trabalham para você — em qualquer ferramenta de IA.**

Diga o que você quer. A IA identifica o squad certo, vira o Chief, vira o agente ideal e executa — tudo em sequência, sem múltiplos agentes rodando ao mesmo tempo.

---

## O que é

Um sistema de 13 squads com agentes especializados que rodam dentro de qualquer ferramenta de IA com acesso a arquivos: Claude Code, Cursor, Windsurf, Codex CLI, Aider, GitHub Copilot.

Inclui:
- **13 squads, 150+ agentes** prontos para usar
- **Base de conhecimento automática** — cada sessão é registrada no Obsidian
- **71 design system templates** de marcas reais (Airbnb, Apple, Stripe, Notion, Cursor...) usados automaticamente em qualquer trabalho de frontend
- **Instalação automática** — uma IA lê o `INSTALAR.md` e configura tudo

## Como Instalar

**Pré-requisitos:** Git, Python 3, Obsidian (https://obsidian.md)

1. Mande sua IA ler o `INSTALAR.md`:
   > "Leia e execute: INSTALAR.md"

2. A IA faz tudo sozinha:
   - Detecta seu home
   - Cria `~/obsidian/` com toda a estrutura
   - Clona este repo e os design templates
   - Configura os hooks automáticos

3. Abra o Obsidian → "Abrir pasta como cofre" → selecione `~/obsidian/`

Pronto.

---

## Como Usar

Após instalado, simplesmente diga o que você quer:

- `"quero criar uma oferta"` → META-CHIEF identifica Hormozi Squad → Chief escolhe o agente → executa
- `"cria o frontend do app"` → Design Squad lê templates de marcas reais → entrega layout autêntico
- `"análise estratégica do negócio"` → Advisory Board com Ray Dalio, Charlie Munger e outros
- `"salva na base"` → ingere na base de conhecimento do Obsidian

## Protocolo de Transformação

```
Pedido do usuário
      ↓
META-CHIEF analisa e identifica o squad
      ↓
Vira o Chief do squad → decide o agente ideal
      ↓
Vira o agente → executa a tarefa completa
```

Uma única IA que se transforma em sequência. Nunca múltiplos agentes em paralelo.
